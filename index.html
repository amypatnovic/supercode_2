Instructions

Learn to code by creating your own filtered photo!





Code Creations

Setup

Tools

  • Download and install Microsoft Visual Studio Code on your computer
  • Open the application
  • Click Extensions icon in the left sidebar and search for GistPad
  • Select and Install

GistPad

  • Create a new playground and name it
  • Under Gists, click on Follow User icon and select [WW84 Coder]
  • From here, you will have access to preset filters to edit your photo

Code Lesson

Code Setup

  • Each Gist listed under user [WW84 Coder] is a separate filter, select the one you want to use by right clicking and selecting Open Gist
  • Copy the HTML and CSS from the filter you choose and paste into your Gist

Photo

  • To upload the photo, right click on the gist and select upload photo
  • In the CSS, replace YOUR PHOTO HERE text with the name of your image

Customize

Colors & Text

  • If your filter uses a text overlay, you can update the message in the HTML window
  • To change color properties, you can do so in the CSS window using hex values

Share

Sharing
Once your image is complete export the file and share on social media
